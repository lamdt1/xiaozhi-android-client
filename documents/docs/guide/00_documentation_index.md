# Android-xiaozhi Documentation Index

This index contains all functional documentation for the Android-xiaozhi project, divided into functional modules for easy reference and use.

The project is developed based on the Flutter framework, supporting iOS, Android, Web, and more, providing rich voice interaction and AI dialogue features.
Visual recognition functions require an API Key to be configured.
